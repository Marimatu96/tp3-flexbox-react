# tp3-flexbox-react

Trabajo práctico de Programación III (UTN La Rioja): Flexbox, formularios dinámicos y React.

## Sitio publicado

https://marimatu96.github.io/tp3-flexbox-react/

## Páginas

- `index.html` → página principal, con enlaces a los 3 ejercicios.
- `flexbox.html` → Ejercicio 1: tarjetas con Flexbox y JavaScript común.
- `personas.html` → Ejercicio 2: formulario y tabla de personas con JavaScript común.
- `react.html` → Ejercicio 3: la misma tabla de personas, resuelta con React.

## Tecnologías

- HTML5 + CSS3 (Flexbox)
- JavaScript común (DOM, eventos, arreglos)
- React 18 (vía CDN, sin herramientas de build) + Babel standalone
